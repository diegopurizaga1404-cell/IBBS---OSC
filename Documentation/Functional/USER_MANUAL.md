# USER MANUAL: IBBS Operations Guide

## 1. Getting Started
To access the IBBS platform, navigate to the main URL and log in with your corporate credentials.

## 2. Navigation
The sidebar on the left provides access to the different modules:
- **Entidades**: Creation, Registry, and Ticket management for service points.
- **SOC Nodos**: Creation and Registry for critical technical nodes.
- **Biblioteca**: Centralized resource and link library.
- **Dashboard**: Real-time monitoring of all incidents.

## 3. Module Guides

### 3.1 Entidades: Creación de Registro (Tab 1)
1. Fill in the "Datos del Solicitante" (Applicant Data).
2. Use the cascade selectors to choose the Region, Province, Locality, and Entity.
3. Click "Finalizar mi registro" to save the incident.

### 3.2 SOC Nodos: Registro (Tab 4)
- **Filters**: Use the top bar to filter by date, region, or status (Pending/Confirmed).
- **Editing**: Administrators can click "Edit Description" to update ticket details inline.
- **Exporting**: Use the "Export to Excel" button to download a filtered report.

### 3.3 Dashboard (SOC Control Center)
- Monitor the **KPI Cards** at the top for real-time ticket counts.
- View the **Charts** for regional trends and entity distribution.
- Use the **Quick Range** buttons (Today, 7d, 30d) to adjust the timeframe.

## 4. Language Settings
Toggle the language between **Español** and **English** using the 🌐 icon in the top bar. All labels, placeholders, and status messages will update automatically.

## 5. Theme Support
Switch between **Light Mode** and **Dark Mode** using the 🌙/☀️ icon at the bottom of the sidebar.
